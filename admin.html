📖 Cách Hoạt Động

Hệ thống này dùng GitHub API để cập nhật products.json trực tiếp lên repo — không cần server!

  1. Tạo repository trên GitHub và bật GitHub Pages (Settings → Pages → Deploy from branch main)
  2. Tạo Personal Access Token: GitHub Settings → Developer Settings → Personal Access Tokens → Fine-grained tokens
  3. Token cần quyền Contents: Read & Write cho repository của bạn
  4. Điền thông tin phía dưới và nhấn Kiểm Tra Kết Nối
  5. Sau đó nhấn 🚀 Đăng Lên GitHub để cập nhật website

🔑 Cấu Hình GitHub

Thông tin này được lưu trong localStorage của trình duyệt, không gửi đi đâu ngoài GitHub API.

Tạo token tại: github.com/settings/tokens

🌐 URL Website Của Bạn

Sau khi push lên GitHub Pages, website sẽ có địa chỉ:

https://[username].github.io/[repo-name]/
